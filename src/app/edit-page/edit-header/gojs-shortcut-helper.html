Shortcuts for GoJS:

Left-click : Select an element

Right-click : Group option reveals if multiple element is selected

Ctrl + Click : Select multiple elements

Space: Positions the selected element to the middle

Delete : Remove selected elements

Ctrl + Mousewheel : You can zoom in/out

Arrowkeys : moves the diagram window

Ctrl + C : Copy selected component

Ctrl + V : Paste element on clipboard

Ctrl + Z : Undo

Ctrl + Y : Redo

Remember to have your focus on the canvas!
